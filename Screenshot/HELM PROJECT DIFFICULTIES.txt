# Main Project

# Phase 6: Create Custom Helm Chart

Created chart structure:

```text
nginx-app/
│
├── Chart.yaml
├── values.yaml
└── templates/
    ├── deployment.yaml
    └── service.yaml
```

Learned:

### Chart.yaml

Stores metadata:

```yaml
apiVersion: v2
name: nginx-app
version: 0.1.0
```

### values.yaml

Stores configurable values:

```yaml
replicaCount: 2

image:
  repository: nginx
  tag: latest
```

### deployment.yaml

Created Deployment template using:

```yaml
{{ .Release.Name }}
{{ .Release.Namespace }}
{{ .Values.replicaCount }}
```

### service.yaml

Created ClusterIP Service.

## Phase 8: Troubleshooting Helm Errors

Solved:

### Error 1

```text
expected at most two arguments
```

Reason:

Incorrect Helm install syntax.

---

### Error 2

```text
repo Jenkins not found
```

Reason:

Repository name mismatch.

---

### Error 3

```text
template: no template nginx-app.fullname
```

Reason:

Leftover Helm-generated files.

Solution:

Removed unnecessary templates.

---

### Error 4

```text
deployment.ymal
```

Fixed typo:

```text
deployment.yaml
```

---

### Error 5

YAML indentation issues in service.yaml.

Corrected:

```yaml
ports:
  - name: http
    port: 81
```
